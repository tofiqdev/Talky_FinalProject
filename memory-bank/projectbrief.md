# Project Brief

## Proje Adı
Talky - Gerçek Zamanlı Mesajlaşma Platformu

## Proje Hedefi
Modern, hızlı ve kullanıcı dostu bir mesajlaşma web uygulaması geliştirmek. Gerçek zamanlı iletişim, kullanıcı yönetimi ve mesaj geçmişi özellikleri sunmak.

## Temel Gereksinimler

### Frontend
- React 19.2.0 + TypeScript
- Tailwind CSS ile modern UI
- Vite build tool
- SignalR client (gerçek zamanlı iletişim)
- Responsive tasarım

### Backend
- .NET 8 C# Web API
- SignalR (WebSocket iletişimi)
- Entity Framework Core
- SQL Server / PostgreSQL
- JWT Authentication
- RESTful API

## Proje Kapsamı

### Temel Özellikler
- Kullanıcı kayıt ve giriş
- Gerçek zamanlı mesajlaşma
- Kişi listesi
- Mesaj geçmişi
- Online/offline durumu
- Yazıyor göstergesi
- Mesaj bildirimleri

### Teknik Gereksinimler
- WebSocket bağlantısı (SignalR)
- JWT token tabanlı kimlik doğrulama
- Database için Entity Framework
- API ve Frontend ayrı portlarda çalışacak

## Başarı Kriterleri
- Kullanıcılar kayıt olup giriş yapabiliyor
- Mesajlar gerçek zamanlı iletiliyor
- Online durumlar güncelleniyor
- Mesaj geçmişi saklanıyor ve yükleniyor
- Responsive tasarım tüm cihazlarda çalışıyor
